# TypeScript SDK

## Installation

```bash
npm install @bkey/sdk
```

## Usage

```typescript
import { BKeyClient } from '@bkey/sdk';

const bkey = new BKeyClient({
  clientId: process.env.BKEY_CLIENT_ID!,
  clientSecret: process.env.BKEY_CLIENT_SECRET!,
});
```
