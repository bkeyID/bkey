# Python SDK

## Installation

```bash
pip install bkey
pip install bkey[async]   # for async support
pip install bkey[all]     # everything
```

## Usage

```python
from bkey import BKeyClient

client = BKeyClient(
    client_id="your-client-id",
    client_secret="your-client-secret",
)
```
