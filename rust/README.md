# bkey

Rust SDK for BKey — biometric approval, vault, and checkout for AI agents.

**This crate is under development.** The API is not yet stable.

See [bkey.id/docs](https://bkey.id/docs) for documentation and [github.com/bkeyID/bkey](https://github.com/bkeyID/bkey) for the full monorepo with TypeScript and Python SDKs.
